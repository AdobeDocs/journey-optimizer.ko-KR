## 작업 캠페인 기본 정보 {#about}

액션 캠페인(또는 예약 캠페인)을 사용하면 프로모션 제안, 참여 캠페인, 공지, 법적 고지 또는 정책 업데이트와 같은 마케팅 사용 사례에 대한 간단한 임시 일괄 커뮤니케이션을 수행할 수 있습니다.
