# 콘텐츠 최적화 시작 {#message-optimization}

>[!CONTEXTUALHELP]
>id="ajo_campaigns_content_optimization"
>title="콘텐츠 최적화"
>abstract="Journey Optimizer의 컨텐츠 최적화를 사용하면 다양한 버전의 컨텐츠를 테스트하고 성과가 가장 좋은 컨텐츠를 결정할 수 있습니다. 타겟팅을 사용하여 특정 세그먼트에 개인화된 콘텐츠를 전달하거나, 여러 변형을 테스트하는 실험 또는, 정교한 최적화 전략을 위해 두 접근 방식을 모두 결합할 수 있습니다."

콘텐츠 최적화는 적절한 시간에 적절한 대상자에게 적절한 메시지를 전달할 수 있는 도구를 제공합니다. 데이터 기반의 통찰력을 강력한 개인화 기능과 결합하여 캠페인 및 여정 전반의 참여와 전환을 극대화할 수 있습니다.

콘텐츠 최적화는 [캠페인](../campaigns/create-campaign.md)과 [여정](../building-journeys/journey-gs.md) 모두에서 사용할 수 있으므로 모든 고객 접점에 동일한 최적화 전략을 적용할 수 있습니다.

➡️ [이 비디오에서 캠페인 내에서 콘텐츠 최적화를 활용하는 방법을 알아보세요](#video)

## 최적화 기능 {#capabilities}

Journey Optimizer의 콘텐츠 최적화를 통해 다음과 같은 작업을 수행할 수 있습니다.

* [타깃팅을 사용](optimization-targeting.md)하여 프로필 특성, 컨텍스트 데이터 또는 대상 멤버십을 기반으로 개인화된 콘텐츠를 특정 대상 세그먼트에 전달하십시오.

* [실험을 실행](optimization-experimentation.md)하여 여러 콘텐츠 변형을 테스트하고 성공 지표를 기반으로 성과가 가장 좋은 콘텐츠를 식별합니다.

* [두 방법을 결합](optimization-combination.md)하여 각 대상 세그먼트에 대해 다양한 변형을 테스트하는 정교한 최적화 전략을 만듭니다.

## 타겟팅 대 실험 {#targeting-vs-experimentation}

타겟팅과 실험의 차이점을 이해하면 목표에 적합한 최적화 접근 방식을 선택하는 데 도움이 됩니다.

**타깃팅**&#x200B;은(는) 결정론적 규칙을 사용하여 알려진 프로필 특성, 컨텍스트 또는 대상 멤버십을 기반으로 개인화된 콘텐츠를 특정 세그먼트에 전달합니다. 이렇게 하면 올바른 메시지가 올바른 대상자에게 도달합니다.

**실험**&#x200B;에서는 무작위 할당을 사용하여 여러 콘텐츠 변형을 테스트하고 성과가 가장 좋은 콘텐츠를 검색합니다. 데이터 기반 테스트를 통해 대상자에게 가장 큰 반향을 불러일으키는 것이 무엇인지 학습하는 데 도움이 됩니다.

아래 표에는 주요 차이점이 요약되어 있습니다.

| 기능 | 타기팅 | 실험 |
|--------|-----------|-----------------|
| **할당 메서드** | 결정론적 - 규칙 기반 | 임의 - 트래픽 할당 기반 |
| **기준** | 프로필 속성, 컨텍스트, 대상자 | 임의 분포 |
| **사용 사례** | 알려진 세그먼트에 관련 콘텐츠 전달 | 성과가 가장 좋은 콘텐츠 검색 |
| **예** | 위치별로 다른 프로모션 표시 | 2개의 제목 줄을 테스트하여 더 많은 열림 수를 확인합니다 |
| **추천 항목** | 규모에 맞는 Personalization | 최적화 및 학습 |

## 일반적인 사용 사례 {#use-cases}

다음은 콘텐츠 최적화를 통해 더 나은 결과를 얻을 수 있는 몇 가지 일반적인 시나리오입니다.

타깃팅:

* **지역 타깃팅** - 고객이 있는 위치를 기반으로 위치별 오퍼를 보냅니다. 예를 들어, 더 추운 지역에서는 겨울 코트를, 더 따뜻한 지역에서는 수영복을 장려합니다.

* **장치 최적화** - 데스크톱 사용자를 위한 데스크톱에 최적화된 레이아웃 및 스마트폰 사용자를 위한 모바일에 최적화된 레이아웃과 같은 장치별 콘텐츠를 제공합니다.

실험:

* **A/B 테스트** - 다양한 이메일 제목 줄, call-to-action 단추 또는 프로모션 오퍼를 테스트하여 전환율이 가장 높은 항목을 알아봅니다.

* **라이프사이클 마케팅** - 신규 고객에 대한 다양한 온보딩 메시지와 기존 고객에 대한 유지 오퍼를 테스트합니다.

조합:

* **고급 세분화** - 충성도 계층을 기준으로 고객을 타깃팅하고 각 계층 내에서 다른 보상 메시지를 테스트하여 모든 세그먼트에 대한 참여를 극대화합니다.

## 시작하기 {#get-started}

콘텐츠 최적화를 시작하려면:

1. **캠페인 또는 여정 만들기**: [캠페인](../campaigns/create-campaign.md) 또는 [여정](../building-journeys/journey-gs.md)을 설정하고 하나 이상의 작업을 추가하십시오.

1. **최적화 방법 선택**:
   * [타깃팅을 사용](optimization-targeting.md)하여 특정 세그먼트에 대한 콘텐츠를 개인화할 수 있습니다.
   * [실험을 사용](optimization-experimentation.md)하여 여러 변형을 테스트합니다.
   * 고급 최적화를 위해 [둘 다 결합](optimization-combination.md)합니다.

1. **콘텐츠 정의**: 최적화 전략을 위한 다양한 콘텐츠 변형을 만듭니다.

1. **활성화 및 모니터링**: [보고서](../reports/campaign-global-report-cja.md)에서 최적화된 캠페인이나 여정을 시작하고 성능을 추적합니다.

## 작동 방식 {#how-it-works}

여정 또는 캠페인이 라이브되면 정의한 기준에 따라 프로필이 평가됩니다. 이러한 평가를 기반으로 각 프로필은 가장 적합한 콘텐츠 버전을 받습니다.

1. **프로필 평가** - 프로필이 캠페인이나 여정에 들어오면 Journey Optimizer에서 해당 특성과 컨텍스트를 평가합니다.

1. **콘텐츠 할당** - 최적화 구성 기반:
   * **타깃팅**&#x200B;의 경우 특정 기준과 일치하는 프로필은 해당 개인화된 콘텐츠를 수신합니다.
   * **실험**&#x200B;의 경우 프로필은 트래픽 할당 설정에 따라 다른 콘텐츠 변형에 임의로 할당됩니다.
   * **조합**&#x200B;의 경우 프로필은 먼저 타깃팅 규칙과 일치한 다음 해당 세그먼트에 대한 실험 변형 중 하나에 임의로 할당됩니다.

1. **성능 추적** - Journey Optimizer은 참여 지표 및 전환 데이터를 자동으로 추적하여 성과가 가장 좋은 콘텐츠를 식별하는 데 도움이 됩니다.

## 사용 방법 비디오 {#video}

작업 또는 API 트리거 캠페인에서 콘텐츠 최적화를 활용하는 방법을 알아봅니다. 하위 대상자를 타기팅하고, 위치별 메시지 베리에이션을 만들고, 대체 콘텐츠를 활성화하고, 단일 캠페인 내에서 여러 실험을 실행하는 방법을 알 수 있습니다. 이 튜토리얼에서는 메시지 일관성을 유지하면서 멀티채널 캠페인을 관리하는 방법도 다룹니다.
