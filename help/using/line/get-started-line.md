# LINE 시작 {#get-started-line}

>[!BEGINSHADEBOX]

**이 페이지의 내용:** Journey Optimizer에서 LINE 메시징의 작동 방식을 이해하여 여정과 캠페인을 통해 LINE 앱에서 고객에게 도달할 수 있도록 합니다.

>[!ENDSHADEBOX]

[!DNL Journey Optimizer]에서 고객의 LINE 앱으로 LINE 메시지를 보냅니다. LINE 편집기에서 메시지를 만들고 개인화하고 미리 볼 수 있습니다.

LINE 메시지는 여정 또는 캠페인에서 만들고 보낼 수 있습니다.

* **여정**&#x200B;에서. 여정을 만들고, LINE 활동을 추가하고, 기본 설정을 정의한 다음 **[!UICONTROL 액션: LINE]** 오른쪽 창으로 이동하여 LINE 메시지의 콘텐츠를 만듭니다. [이 페이지](../building-journeys/journey-gs.md)에서 여정을 만드는 방법을 알아봅니다.

* **캠페인**&#x200B;에서. 캠페인을 만들고 액션으로 LINE을 선택하고 기본 설정을 정의한 다음 메시지 콘텐츠를 편집하여 보낼 LINE 메시지를 정의합니다. [액션 캠페인](../campaigns/campaign-action.md#action-campaign-action) | [API 트리거 캠페인](../campaigns/api-triggered-campaigns.md) | [오케스트레이션된 캠페인](../orchestrated/create-orchestrated-campaign.md#create) 만드는 방법 알아보기

LINE 메시지를 만들기 전에 Journey Optimizer에서 LINE 채널을 구성합니다. [자세히 알아보기](line-configuration.md).
