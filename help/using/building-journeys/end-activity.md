**의도:**

* 종료 활동을 추가하여 여정 경로의 끝을 표시합니다
* 종료 활동에 레이블을 지정하여 여정 보고서 가독성 개선
* 레이블이 지정된 종료 노드를 사용하여 시각적으로 여러 여정 경로 구분

**용어집:**

* **종료 활동**: 여정 경로의 끝을 시각적으로 표시하는 여정 노드입니다. 필수 사항은 아니지만 명확성 *(제품별)*&#x200B;에 도움이 됩니다.

**보호 기능:**

* 종료 활동은 필수가 아닙니다. 여정은 종료 활동 없이 작동합니다.
* 라이브 보고서를 더 쉽게 해석할 수 있도록 여정에 여러 경로가 있는 경우 각 종료 활동에 레이블을 추가하는 것이 좋습니다.

**용어:**

* 정식 이름: 종료 활동 — 약어: 해당 사항 없음 — 변형: 종료 노드, 종료 태그
* 혼동하지 마십시오: &quot;종료 활동&quot; ≠ &quot;종료 태그&quot; — 종료 태그는 여정 작성 중에 표시되는 자동 생성 노드이며, 종료 활동은 배치 가능한 구성 요소입니다

**FAQ:**

* **Q: 여정을 게시하는 데 종료 활동이 필요합니까?** — 아니요, 필수는 아니지만 시각적 명확성을 위해 권장됩니다.
* **Q: 종료 활동에 레이블을 지정해야 하는 이유는 무엇입니까?** — 여정에 여러 경로가 있는 경우 여정 보고서에서 레이블을 구별하는 데 도움이 됩니다.
* **Q: 종료 활동 레이블의 영향을 어디에서 확인할 수 있습니까?** — 여정 라이브 보고서에 레이블이 표시되어 경로 수준 성능을 더 쉽게 추적할 수 있습니다.
