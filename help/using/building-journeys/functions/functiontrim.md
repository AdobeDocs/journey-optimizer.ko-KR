# trim {#trim}

시작 및 종료 공백을 제거합니다.

## 카테고리

문자열

## 함수 구문

`trim(<parameters>)`

## 매개 변수

| 매개 변수 | 유형 |
|-----------|------------------|
| string | string |

## 서명 및 반환된 형식

`trim(<string>)`

문자열을 반환합니다.

## 예

`trim(" Hello ")`

&quot;Hello&quot;를 반환합니다.
