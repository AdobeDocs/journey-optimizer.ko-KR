# length {#length}

매개 변수에서 문자열 식의 문자 수를 반환합니다.

## 카테고리

문자열

## 함수 구문

`length(<parameters>)`

## 매개 변수

* string

## 서명 및 반환된 형식

`length(<string>)`

정수를 반환합니다.

## 예

`length("Hello World")`

11을 반환합니다.
