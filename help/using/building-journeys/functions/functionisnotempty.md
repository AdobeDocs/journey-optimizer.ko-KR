# isNotEmpty {#isNotEmpty}

매개 변수의 문자열이 비어 있지 않으면 true를 반환합니다.

## 카테고리

문자열

## 함수 구문

`isNotEmpty(<parameters>)`

## 매개 변수

* string

## 서명 및 반환된 형식

`isNotEmpty(<string>)`

부울을 반환합니다.

## 예

`isNotEmpty("")`

false를 반환합니다.

`isNotEmpty("hello")`

true를 반환합니다.
