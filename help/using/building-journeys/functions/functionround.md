# round {#round}

인수에 가장 가까운 정수 값을 반환하고 양수 무한대로 반올림합니다.

## 카테고리

계산

## 함수 구문

`round(<parameters>)`

## 매개 변수

* 십진수
* 정수

## 서명 및 반환된 유형

`round(<decimal>)`

`round(<integer>)`

정수를 반환합니다.

## 예

`round(3.14)`

3 반환.

`round(3.54)`

4 반환.

`round(-3.14)`

-3을 반환합니다.

`round(3)`

3 반환.
