# isEmpty {#isEmpty}

매개 변수의 문자열에 문자가 없으면 true를 반환합니다.

## 카테고리

문자열

## 함수 구문

`isEmpty(<parameters>)`

## 매개 변수

* string

## 서명 및 반환된 형식

`isEmpty(<string>)`

부울을 반환합니다.

## 예

`isEmpty("")`

true를 반환합니다.

`isEmpty("Hello World")`

false를 반환합니다.
