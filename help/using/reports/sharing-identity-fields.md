# journeyStep 이벤트 ID 필드 {#sharing-identity-fields}

이 필드 그룹은 journeyStepEvent에만 사용됩니다. 이 이벤트는 journey와 관련이 있으며 프로필 ID가 있는 경우 identityMap이 없습니다.

journeyStepEvent의 경우 ID와 관련된 필드를 추가해야 합니다.

## profileID

프로필 식별자

유형: string

## profileNamespace

프로필 식별자 네임스페이스

유형: string
