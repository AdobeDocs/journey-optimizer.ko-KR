# 레거시 필드 정보 {#sharing-legacy-fields}

>[!BEGINSHADEBOX]

* [journeyStep 이벤트 데이터 가져오기 필드](../reports/sharing-fetch-fields.md)
* [journeyStep 이벤트 ID 필드](../reports/sharing-identity-fields.md)
