# 개체 함수 {#objects}

## null임{#isNull}

다음 `isNull` 함수는 개체 참조가 존재하지 않는지 확인합니다.

**구문**

```sql
{%= isNull(object) %}
```

**예**

다음 작업은 개인의 집 주소가 존재하지 않는지 확인합니다.

```sql
{%= isNull(person.homeAddress) %}
```

## null이 아님{#isNotNull}

다음 `isNotNull` 함수는 개체 참조가 있는지 여부를 결정합니다.

**구문**

```sql
{%= isNotNull(object) %}
```

**예**

다음 작업은 개인의 집 주소가 있는지 확인합니다.

```sql
{%= isNotNull(person.homeAddress) %}
```
