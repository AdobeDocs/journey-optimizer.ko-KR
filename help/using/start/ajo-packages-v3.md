# Adobe Journey Optimizer 패키지 및 기능 {#ajo-packages}

[!DNL Adobe Journey Optimizer]은(는) 모듈식 패키징 모델을 사용합니다. 기본 오케스트레이션 사용 사례에 맞는 기본 오퍼로 시작한 다음 필요한 채널 및 고급 기능을 추가합니다.

>[!NOTE]
>
>기능 가용성은 사용권 계약, 선택한 추가 기능, 지역 가용성, 채널 구성 및 구현 사전 요구 사항에 따라 달라질 수 있습니다. 조직 고유의 자세한 내용은 Adobe 담당자에게 문의하십시오.

## 1단계 — 기본 오퍼 선택 {#base-offers}

세 가지 기본 오퍼를 사용할 수 있습니다. 주로 고객을 유도하는 방식에 가장 적합한 것을 선택하십시오.

| 기본 오퍼 | 다음에 최적 | 포함 사항 |
|-----------|---------|-----------------|
| **Journey Optimizer - 캠페인** | 일괄 처리, 마케터 계획 지원 | 배치 참여 및 마케터 계획 전달을 위한 대상 기반, 예약된 캠페인 오케스트레이션. |
| **Journey Optimizer - 여정** | 실시간 고객 참여 | 실시간 고객 참여를 위한 이벤트 중심의 1:1 여정 오케스트레이션. |
| **Journey Optimizer - 캠페인 및 여정** | 두 가지 모두를 필요로 하는 고객 | 하나의 오퍼에 대상 기반 캠페인 오케스트레이션과 실시간 여정 오케스트레이션이 모두 포함됩니다. |

### 캠페인과 여정 비교 — 차이점은 무엇입니까? {#campaigns-vs-journeys}

| 기능 | Journey Optimizer - 캠페인 | JOURNEY OPTIMIZER - 여정 | Journey Optimizer - 캠페인 및 여정 |
|--|:-----------------------------:|:----------------------------:|:----------------------------------------:|
| 대상 기반 일괄 처리 오케스트레이션 | ✓ | — | ✓ |
| 실시간 이벤트 기반 오케스트레이션 | — | ✓ | ✓ |
| 채널 추가 기능 사용 가능 | ✓ | ✓ | ✓ |
| 의사 결정 추가 기능 사용 가능 | ✓ | ✓ | ✓ |

### 언제 캠페인을 선택해야 합니까? {#when-to-choose-campaigns}

기본 사용 사례가 다음과 같은 경우 **Journey Optimizer - 캠페인**&#x200B;을 선택하세요.

- 예약된 캠페인
- 대상자 기반 지원
- 일괄 마케팅 워크플로우
- 마케터가 계획한 참여 프로그램

### 언제 여정을 선택해야 합니까? {#when-to-choose-journeys}

기본 사용 사례가 다음과 같은 경우 **Journey Optimizer - 여정**&#x200B;을(를) 선택하십시오.

- 이벤트 트리거된 통신
- 이벤트 후 후속 흐름
- 온보딩 및 라이프사이클 여정
- 실시간 1:1 참여

### 언제 캠페인 및 여정을 선택해야 합니까? {#when-to-choose-both}

다음 두 가지가 모두 필요한 경우 **Journey Optimizer - 캠페인 및 여정**&#x200B;을 선택하세요.

- 대상자 기반 오케스트레이션 일괄 처리 및
- 실시간 이벤트 기반 오케스트레이션

## 자주 묻는 질문 {#faq}

+++**모든 기본 오퍼에 모든 채널이 포함되어 있습니까?**

아니요. 기본 오퍼는 오케스트레이션 기능을 정의합니다. 채널 액세스는 조직의 추가 기능 및 라이센스 구성에 따라 다릅니다.

+++

+++**캠페인과 여정 간의 차이점은 무엇입니까?**

**캠페인**&#x200B;은(는) 대상을 기반으로 하며 마케터가 계획합니다. 이는 예약된 봉사, 홍보 프로그램 및 일괄 캠페인 워크플로우에 가장 적합합니다.

**여정**&#x200B;은(는) 실시간 및 이벤트를 기반으로 합니다. 이는 고객 행동에 반응하는 1:1 참여에 가장 적합합니다.

**캠페인 및 여정**&#x200B;에 오케스트레이션 모델이 모두 포함되어 있습니다.

+++

+++**오케스트레이션된 캠페인에서 지원되는 채널은 무엇입니까?**

오케스트레이션된 캠페인은 캠페인 오케스트레이션 워크플로우에 사용되는 채널의 하위 집합을 지원합니다. 채널 가용성은 사용 허가된 구성에 따라 다릅니다. [오케스트레이션된 캠페인에 대해 자세히 알아보기](../orchestrated/gs-orchestrated-campaigns.md)

+++

+++**의사 결정이 모든 구성에 포함되어 있습니까?**

아니요. Decisioning은 별도의 고급 기능 추가 기능입니다. [의사 결정 사용 방법 알아보기](../experience-decisioning/gs-experience-decisioning.md)

+++

+++**Select, Prime 또는 Ultimate에 대해 들었습니다. 아직 현재 패키징 모델입니까?**

[!DNL Adobe Journey Optimizer]은(는) 기본 오퍼 및 추가 기능을 기반으로 구축된 모듈식 패키징 모델을 통해 제공됩니다. 기존 패키징 용어를 사용하는 기존 고객의 경우 Adobe 담당자에게 문의하여 현재 사용 권한을 확인하십시오.

+++
