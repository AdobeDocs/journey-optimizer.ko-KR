# Journey Optimizer 기능 가용성 {#ajo-features-availability}

>[!BEGINSHADEBOX]

**이 페이지에서 사용할 수 있는 [!DNL Adobe Journey Optimizer] 기능, 각 기능이 속한 라이프사이클 단계(일반 가용성, 제한된 가용성 또는 Beta)가 무엇인지, 해당 기능이 적용되는 기본 오퍼 및 제공된 시기를 확인하십시오. *에 답할 수 있습니다.&quot;이 기능을 사용할 수 있습니까?&quot;***&#x200B;릴리스 노트를 자세히 살펴보지 않고

>[!ENDSHADEBOX]

이 페이지에서는 [!DNL Adobe Journey Optimizer]의 기능 가용성을 통합하므로 구현 전 범위 지정 중에 사용할 수 있는 기능을 확인할 수 있습니다. 기능은 기능 영역별로 그룹화됩니다. 각 영역 내에서 모든 기능에는 현재 라이프사이클 상태, 적용되는 기본 오퍼, 사용 가능한 날짜 및 구성 또는 지역 제한에 대한 참고 사항이 나열됩니다.

*다음 이후에 사용 가능* 열에서 **핵심 기능**&#x200B;으로 표시된 행은 2026년 이전부터 일반적으로 사용할 수 있는 오래 지속되는 기본 기능입니다. 일자 행은 2026년에 제공된 변경 사항을 반영합니다.

>[!IMPORTANT]
>
>**내 환경에서 기능을 볼 수 없는 이유는 무엇입니까?** **제한된 가용성** 또는 **Beta**&#x200B;의 기능은 모든 사용자에게 표시되지 않습니다. 이러한 기능은 먼저 제한된 조직 집합으로 롤아웃됩니다. 읽은 기능이 내 환경에 표시되지 않으면 아래 상태를 확인하십시오. **LA** 또는 **Beta**&#x200B;인 경우 Adobe 담당자에게 문의하여 액세스를 요청하십시오. 여기에 나열된 기능은 사용 중인 환경에서 활성화되었음을 의미하지 않습니다.

>[!NOTE]
>
>**사용 가능 여부 대 권한** 이 페이지에서는 *기능 수명 주기 및 가용성*&#x200B;을 추적합니다(기능의 출시 여부 및 출시 시기). 라이선스에 *기능이 포함되어 있는지 여부*&#x200B;는 기본 오퍼 및 추가 기능에 따라 다릅니다. [패키지 및 기능](ajo-packages.md)을 참조하세요.

| 상태 | 의미 |

| **LA** — 제한된 가용성 | 제한된 조직 집합에 릴리스됩니다. 액세스를 요청하려면 **Adobe 담당자에게 문의**&#x200B;하십시오. |

## &quot;적용 대상&quot;이 기본 오퍼에 매핑되는 방법 {#applies-to}

**적용 대상** 열은 세 개의 [!DNL Adobe Journey Optimizer] 기본 오퍼를 참조합니다.

- **Journey Optimizer - 캠페인** — 일괄 처리, 대상 기반 오케스트레이션
- **Journey Optimizer - 여정** — 실시간, 이벤트 기반 오케스트레이션
- **Journey Optimizer - 캠페인 및 여정** — 둘 다

**모든 기본 오퍼**&#x200B;로 표시된 채널, 콘텐츠 및 플랫폼 기능은 기본 오퍼에 관계없이 사용할 수 있지만 대부분은 여전히 관련 채널 또는 고급 기능 추가 기능이 필요합니다. 권한 부여에 대한 자세한 내용은 [패키지 및 기능](ajo-packages.md)을 참조하세요.
