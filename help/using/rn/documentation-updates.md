# 설명서 업데이트 {#latest-updates}

## 2022년 12월 {#december-2022}

   * [이메일 채널](../email/get-started-email.md)
   * [푸시 알림 채널](../push/get-started-push.md)

## 2022년 11월 {#november-2022}

* Journey Optimizer 통합에 대한 새 페이지가 추가되었습니다. [자세한 내용](../start/ajo-integrations.md)
* 미러 페이지 URL 길이에 대한 권장 사항을 추가했습니다. [자세한 내용](../email/message-tracking.md)

## 2022년 10월 일 {#october-2022}

* 외부 데이터 소스 및 사용자 지정 작업을 사용하여 처리량을 제한하는 방법에 대한 여정 사용 사례를 추가했습니다. [자세한 내용](../building-journeys/limit-throughput.md)
* 여정 사용 사례 섹션이 다음 두 가지 카테고리로 재구성되었습니다. [비즈니스 사용 사례](../building-journeys/journeys-uc.md) 및 [기술 사용 사례](../building-journeys/collections.md).
* 다음 **엔티티 데이터 세트** 섹션을 더 자세히 업데이트했습니다. [자세한 내용](../data/datasets-query-examples.md#entity-dataset)
* 옵트아웃 관리 및 동의 정책 섹션이 재구성되었습니다. [자세한 내용](../privacy/opt-out.md)
* 여정 메시지의 고급 매개 변수 섹션을 보다 명확하게 수정했으므로 이제 전자 메일 주소 무시를 특정 사용 사례에만 사용하도록 지정합니다. 대부분의 경우 **실행 필드** 는 사용해야 하는 것입니다.
* 에 메모를 추가했습니다. **랜딩 페이지 하위 도메인 구성** 섹션에 랜딩 페이지 하위 도메인에서 대문자를 사용할 수 없음을 지정합니다. [자세한 내용](../landing-pages/lp-subdomains.md)

* Adobe Journey Optimizer에서 사용할 수 있는 모든 보고 지표 목록이 추가되었습니다.
