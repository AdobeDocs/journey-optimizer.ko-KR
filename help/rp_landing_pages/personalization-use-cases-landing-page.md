# 개인화 사용 사례{#section-overview}

이 섹션에서는 Adobe Journey Optimizer에서 개인화된 메시지를 작성하는 단계별 사용 사례를 제공합니다. 각 예시는 실제 시나리오를 기반으로 하며, 프로필 속성, 상황별 데이터, 도우미 함수 및 오퍼 결정을 결합하여 대상자에게 타겟팅되고 관련성 높은 콘텐츠를 제공하는 방법을 보여줍니다.

## 개인화 사용 사례

:::
![icon](https://cdn.experienceleague.adobe.com/icons/circle-play.svg?lang=ko)

주문 상태 알림

프로필 속성, 실시간 상황별 데이터, 오퍼 결정을 결합하여 고객에게 주문 상태를 확인하거나 업데이트하는 개인화된 푸시 알림을 보내는 전체 시나리오를 살펴봅니다.

[튜토리얼 보기](../using/personalization/personalization-use-case.md)
:::

:::
![icon](https://cdn.experienceleague.adobe.com/icons/envelope.svg?lang=ko)

장바구니 포기 이메일

도우미 함수를 사용하여 고객별 세부 정보와 제품 정보를 장바구니 포기 이메일에 동적으로 입력하는 방법을 알아봅니다. 관련성 높고 시기적절한 메시지로 매출 손실을 만회할 수 있습니다.

[안내서 읽기](../using/personalization/personalization-use-case-helper-functions.md)
:::

:::
![icon](https://cdn.experienceleague.adobe.com/icons/list-check.svg?lang=ko)

건강 보험 처방전 경고

조건부 논리와 템플릿 표현식을 사용하여 구독자의 모든 처방전이 현재 &quot;수령 준비 완료&quot; 또는 &quot;리콜&quot; 상태인지 나열하는 개인화된 이메일을 한 번에 작성합니다.

[예시 읽기](../using/personalization/perso-uc-plan-prescriptions.md)
:::

:::
![icon](https://cdn.experienceleague.adobe.com/icons/code-branch.svg?lang=ko)

개인화 레시피

날짜, 프로필 속성, 조건부 논리 등을 포함하는 바로 사용할 수 있는 개인화 패턴 라이브러리를 살펴봅니다. 각 레시피에는 이메일, SMS 또는 푸시 알림 콘텐츠에 직접 복사할 수 있는 구문 스니펫이 포함되어 있습니다.

[레시피 찾아보기](../using/personalization/personalization-recipes.md)
